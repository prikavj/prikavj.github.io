# Images Directory

This folder contains all images used in the portfolio website.

## Folder Structure

### `profile/`
Place your profile picture here.
- **Current filename:** `me-big.png` (already in use)
- **Alternative filenames:** `profile.jpg` or `profile.png` (update code if using these)
- **Recommended size:** 400x400px or larger (square aspect ratio works best)
- **Formats:** JPG, PNG, or WebP

### `companies/`
Place company logos/icons here. Use the company name as the filename.
- **Vector Institute:** `vector-institute.png` or `vector-institute.svg`
- **Social & Intelligent Robotics Research Lab:** `sirrl.png` or `sirrl.svg`
- **HPE Aruba Networking:** `hpe.png` or `hpe.svg`

**Recommended format:** SVG (scalable) or PNG with transparent background
**Recommended size:** 200x200px or larger

## Usage

Images in the `public/` folder can be referenced directly in your code:
- Profile: `/images/profile/profile.jpg`
- Companies: `/images/companies/vector-institute.png`

The code will automatically handle these paths.
